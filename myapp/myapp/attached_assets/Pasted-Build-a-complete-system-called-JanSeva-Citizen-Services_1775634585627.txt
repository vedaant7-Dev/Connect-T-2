Build a complete system called "JanSeva – Citizen Services Platform" with a mobile app and a web-based admin panel.

SYSTEM STRUCTURE:

* Mobile App (Flutter) for Users and Nagarsevak
* Web Admin Panel (React or Next.js) for Head Admin
* Shared backend (Firebase or Node.js) with real-time database

ROLES:

1. Citizen User (mobile app)
2. Nagarsevak (mobile app, ward-based access)
3. Head Admin (web panel with full control)

MOBILE APP FEATURES:

* User login via OTP and Google
* Submit complaints with image, category, and auto location
* Track complaint status in real-time
* View dynamic service categories (hospital, school, shamshanbhumi, etc.)
* Click service → view details and multiple contact numbers
* Community feed (posts, likes, comments)
* Live chat with moderation
* Notifications and alerts

NAGARSEVAK FEATURES:

* Login in mobile app
* Access only complaints from assigned ward
* Update complaint status and upload proof
* Cannot access other wards

WEB ADMIN PANEL FEATURES:

DASHBOARD:

* Total complaints statistics
* Ward-wise complaint data
* Nagarsevak performance tracking

COMPLAINT MANAGEMENT:

* View all complaints across wards
* Filter by ward, status, and date
* View assigned officer and resolution timeline

NAGARSEVAK MANAGEMENT:

* Add, edit, delete nagarsevak users
* Assign wards

SERVICE MANAGEMENT (DYNAMIC SYSTEM):

* Create, edit, delete service categories
* Add, edit, delete services under categories
* Each service includes:

  * Name
  * Address
  * Location (map)
  * Multiple contact numbers
  * Optional WhatsApp
  * Optional images
  * Optional religion/category (for shamshanbhumi)

COMMUNITY FEED:

* Admin can create official posts
* Moderate and delete user posts

LIVE CHAT MODERATION:

* Detect abusive or sexual language using AI
* Block users temporarily or permanently

NOTIFICATIONS:

* Send broadcast alerts to all users

TECH REQUIREMENTS:

* Flutter mobile app
* React/Next.js web panel
* Firebase or Node.js backend
* Firestore or MongoDB database
* Google Maps integration
* Real-time sync between app and web panel

UI:

* Clean, modern design
* Blue gradient theme
* Card-based layout
* Multilingual support (English, Marathi, Hindi)

Ensure all data updates from the web admin panel reflect instantly in the mobile app.
