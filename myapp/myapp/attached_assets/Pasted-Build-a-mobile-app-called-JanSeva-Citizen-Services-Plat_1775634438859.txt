Build a mobile app called "JanSeva – Citizen Services Platform" with a dynamic service management system.

ROLES:

1. Citizen User
2. Nagarsevak (Ward Admin)
3. Head Admin (Super Admin)

CORE SYSTEM:
The app must not use fixed services. Instead, implement a dynamic service system where the Head Admin can create, update, or delete any type of service.

SERVICE CATEGORY SYSTEM:

* Admin can create service categories such as Hospital, Clinic, School, Shamshanbhumi, Post Office, Police Station, etc.
* Each category has name and icon.

SERVICE MANAGEMENT:

* Admin can add, edit, or delete services under any category.
* Each service must include:

  * Name
  * Category
  * Description
  * Address
  * Latitude and Longitude (for maps)
  * Multiple contact numbers (array)
  * Optional WhatsApp number
  * Open/Close timings
  * Optional images
  * Optional religion/category (for Shamshanbhumi: Hindu, Muslim, Buddhist, Christian)

USER FEATURES:

* Login via OTP or Google
* View all service categories dynamically on home screen
* Click category → view nearby services
* Click service → view full details and multiple contact buttons (call/WhatsApp)
* Submit and track complaints
* Access community feed
* Use live chat

NAGARSEVAK FEATURES:

* Login with ward-based access
* View only complaints from assigned ward
* Update complaint status and upload proof

HEAD ADMIN FEATURES:

* Dashboard showing all complaints across wards
* Monitor which nagarsevak is handling each complaint
* Manage all service categories and services (add/edit/delete)
* Manage users and nagarsevaks

COMMUNITY FEED:

* Admin and users can post updates
* Like, comment, share
* Admin moderation system

LIVE CHAT:

* Real-time chat
* AI-based filtering for abusive or sexual language
* Temporary or permanent user blocking system

NOTIFICATIONS:

* Complaint updates
* Service updates
* Alerts and schemes

TECH STACK:

* Flutter frontend
* Firebase backend (Firestore for real-time)
* Google Maps API for location services
* AI moderation for chat

UI DESIGN:

* Clean modern UI
* Blue gradient theme
* Card-based layout
* Multilingual support (English, Marathi, Hindi)
